JSON properties must use camelCase.

Schema/object names must use PascalCase.

JSON fields must not use ALL_CAPS, snake_case, or kebab-case.

Names must be descriptive and unambiguous.

Avoid abbreviations unless they are industry standard.

Fields ending with Id must represent unique identifiers.

Identifier fields must use the suffix Id exactly.

Identifier fields should be strings unless explicitly numeric.

Arrays must use plural names.

Object names must be singular.
